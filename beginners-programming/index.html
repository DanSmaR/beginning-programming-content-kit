Beginners programming TopicKit

Beginning your journey towards learning to code

Hello and welcome to the "Beginners programming" TopicKit. This kit contains resources that you can use the learn and teach general programming principles and the JavaScript language. The material is divided into three sections:

  1. Guides: This section contains a full 8-part tutorial series tat you can work through on your own, or use for collaborative learning or teaching a class. This section also contains a number of demos to demonstrate the topics in action.
  2. Slides: This section contains a slide deck that can be used to give a presentation on the main topic of the TopicKit.
  3. Slide script: This section contains notes to go along with the slide deck. We have kept these separate from the slides so they are easier to translate.

Translations

No translations as yet. If you are interested in submitting one, please add it to the l10n directory in the repo, and submit a pull request.
